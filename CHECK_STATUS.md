https://johnkibre.github.io/test.html

### 4. Deployment Status
https://github.com/johnkibre/johnkibre.github.io/deployments

**What you should see:**

- Red X = failed deployments
- Nothing = GitHub Pages not enabled

## Most Common Issue:

**GitHub Pages is NOT enabled yet!**

To enable:
1. Go to: https://github.com/johnkibre/johnkibre.github.io/settings/pages
2. Under "Source" select "Deploy from a branch"
3. Under "Branch" select "main" and "/ (root)"
4. Click "Save"
5. Wait 3 minutes
6. Try again!
